# How to compile
### Cloning Repository
Clone the repo using `git`, and change your directory into the root of the project.  

```console
$ git clone https://github.com/yoshisaac/CounterStrike2-Linux-Cheat.git  
$ cd CounterStrike2-Linux-Cheat/
```
